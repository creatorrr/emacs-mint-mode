emacs mint mode
===============

Major mode for [mint lang](https://mint-lang.com).

Features
--------

 - Syntax highlighting
 - Auto format on save using `mint format`

Planned roadmap
---------------

 - Autocomplete support
 - Lint css values
 - Highlight HTML tags and attributes
 - Emacs shell commands for using the mint CLI
 - Documentation search
